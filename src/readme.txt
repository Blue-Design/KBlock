KBlock是基于MIT媒体实验室的开源Scratch代码开发的图形化编程软件。


KBlock在Scratch的基础上，增加一些与硬件相关的积木块，结合咔嗒爸爸可编程控制器，这些积木块可以实现电子模块之间的交互，例如：读取传感器、控制电动机、控制舵机甚至机器人。


了解更多请关注咔嗒爸爸官方网站：http://www.kadapapa.com/


咔嗒爸爸积木块内容：

程序头
1.KBlock主程序

输出部分
1.设置接口数字输出
2.设置接口PWM输出
3.设置LED状态
4.设置振动器状态
5.设置数码管显示
6.设置RGB颜色
7.设置舵机角度
8.设置电机转速
9.设置蜂鸣器
10.计时器归零


模拟输入部分
1.读取接口模拟值
2.光线传感器
3.声音传感器
4.火焰传感器
5.震动传感器
6.温度传感器
7.湿度传感器
8.红外接收器
9.陀螺仪
10.计时器


数字输入部分
1.读取接口数字量
2.光电开关
3.限位开关
4.按钮开关
5.人体接近开关
6.四位按键
7.红外遥控
8.高电平
9.低电平
10.打开
11.关闭
